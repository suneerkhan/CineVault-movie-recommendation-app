# 🎬 CineVault — Movie Discovery App

A sleek, cinema-themed movie discovery web app built with vanilla HTML, CSS, and JavaScript. Search 500,000+ movies, manage your watchlist, and explore popular titles — all powered by the OMDB API.

## ✨ Features

- 🔍 **Live Search** — instant suggestions dropdown as you type
- 🎬 **Movie Details** — full modal with plot, cast, director, awards, box office
- 🔖 **Watchlist** — save movies locally, persists across sessions
- 🔥 **Now Popular Strip** — horizontally scrollable strip of trending movies
- 🎨 **Light / Dark Theme** — toggle with persistence
- 📱 **Fully Responsive** — works on mobile, tablet, and desktop
- 🧩 **Filters** — Genre, Year, Language, Rating, Sort
- ⊞ **Grid / List View** — switch between layouts
- ⬆️ **Scroll to Top** button
- 💡 **Cursor Spotlight** — subtle gold glow follows your cursor
